Sharedsqlite.

Handling persistent data in android can sometimes be a headache especially when you only want to Store a single row (that of course if you wish to use the same value later on), The system provides a SharedPreferences Object however it has some 'dirty job' around it...

Introducing - Sharedsqlite!

Sharedsqlite is a simple thread-safe database with a single table with a key-value schema where you can save your (Mainly) primitive data types that suites nowhere else excluding the headache of messing around with committing and reading From the sharedPreferences.

Jar File

Jar file can be downloaded right here Just add the jar to your project 'libs' folder and you're good to go.

Usage

Writing and reading the data is only a one line code. First create a new instance by calling - SharedSqlite.initialize(getApplicationContext());

And now you are free to call -
SharedSqlite.getInstance().addValue(key, value)
Wherever you want in your code in order to write the key-value pair to the database When key and value can be any primitive data type or an enumerator (it will turn in to String value in runtime)

In order to read the data later on you can call any of the -

  • SharedSqlite.getInstance().getBooleanValue(Key, DefaultValue); - To read a boolean value or the default value provided in case there is no mapping for the supplied key.

  • SharedSqlite.getInstance().getIntValue(Key, DefaultValue); - To read a int value or the default value provided in case there is no mapping for the supplied key.

  • SharedSqlite.getInstance().getDoubleValue(Key, DefaultValue); - To read a double value or the default value provided in case there is no mapping for the supplied key.

  • SharedSqlite.getInstance().getFloatValue(Key, DefaultValue); - To read a float value or the default value provided in case there is no mapping for the supplied key.

  • SharedSqlite.getInstance().getStringValue(Key, DefaultValue); - To read a String value or the default value provided in case there is no mapping for the supplied key.

  • SharedSqlite.getInstance().getLongValue(Key, DefaultValue); - To read a long value or the default value provided in case there is no mapping for the supplied key.
